Probability Engine
Use the built-in heuristic, then override or extend it.
Formula 1 — Binary arb
Edge = 1 - (YES + NO). If the total is below 1.00, the pair may be structurally underpriced.
Formula 2 — Bucket underpricing
Edge = 1 - Σ(bucket YES prices). This only matters when buckets are actually a complete, non-overlapping set.
Formula 3 — Model edge
Edge = fair probability - market probability. The built-in model starts from market price, then adjusts for liquidity, cheap pricing, time left, and arb strength.

Settings & Notes
This app is a scanner, not an auto-trader.
This patched build tries to be more reliable by parsing token IDs safely whether Polymarket returns them as arrays or JSON-like strings, and by using batch requests where possible. If GitHub Pages shows an old version, hard refresh the page or remove and re-add the Home Screen app.
How this scanner works
Simple scan flow
1) Pull active markets.

2) Parse YES/NO token IDs safely.

3) Fetch prices and midpoints in batches.

4) Score each market for binary arb, model edge, and liquidity quality.

5) Group similar range markets for bucket checks.

6) Save your fair-value overrides and watchlist in your browser.
