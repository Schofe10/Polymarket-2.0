# Polymarket Edge Finder

This is a GitHub Pages-ready static app.

## Upload to GitHub
1. Create a public repo.
2. Upload every file in this folder to the repo root.
3. Go to **Settings → Pages**.
4. Set **Deploy from a branch → main → /(root)**.
5. Open the GitHub Pages URL in Safari.
6. Tap **Share → Add to Home Screen**.

## What this package includes
- Binary YES + NO arbitrage scanner
- Bucket mispricing scanner
- Liquidity gap detector
- Heuristic probability model engine
- Manual fair % overrides
- Optional custom JavaScript model hook
- Watchlist and local save

## Important
This is a scanner only. It does not place trades.
